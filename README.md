# iSAC
implementation of iSAC

# how to use

## clone source code
```sh
git clone https://github.com/Taiki-Ishigaki/iSAC.git iSAC
```

## build sorce code
```sh
cd iSAC
mkdir build
cd build
cmake ..
make
```
